Bases y Condiciones

Bonos y transacciones

Bonos

  • Los bonos se acreditan como fichas de bono y no son retirables de forma directa.
  • El bono queda bloqueado y se habilita cuando el usuario deposita el mismo monto.
  • Las fichas de bono pueden utilizarse para jugar en mesas habilitadas.

Depósitos

  • Las transferencias se validan manualmente.
  • La acreditación puede demorar hasta 2 horas desde la confirmación del envío.
  • Si hay inconsistencias, soporte puede solicitar información adicional.

Código de invitación

  • El bono por invitación se activa con el primer depósito válido.
  • El bono se acredita en fichas bloqueadas y se habilita al depositar el mismo monto.
